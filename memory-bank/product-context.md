# Описание продукта

## Цель проекта
Реализация небольшого web-app криптокошелька (self-custodial) для TON testnet без собственного backend.

## Основной функционал
1. **Создание / импорт кошелька:** (testnet, 24 слова мнемоники).
2. **Главный экран:** Адрес, баланс, список транзакций, поиск по истории.
3. **Получение средств (Receive):** Показ адреса, копирование (QR-код - бонус).
4. **Отправка средств (Send):** Форма с валидацией, отправка с обратной связью (Async Handling).
5. **Защита от подмены адресов (Spoofing Protection):** Анализ вводимого адреса на совпадения со старыми транзакциями (атака нулевыми переводами) с выдачей предупреждений.

## Ограничения и Условия
- **Тестовая сеть:** Исключительно TON testnet.
- **Хранение:** Данные для UX хранятся локально в браузере (localStorage/memory).
- **Self-custodial:** Отсутствие бэкенда, генерация ключей и подписание транзакций происходят на клиенте.
- **UI:** Минималистичный, 100% pixel-perfect (сделано), с заметными предупреждениями безопасности.
